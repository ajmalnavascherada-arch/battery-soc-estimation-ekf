# 🔋 Battery State of Charge Estimation using EKF

## 📌 Overview

This project estimates the **State of Charge (SoC)** of a lithium-ion battery using an **Extended Kalman Filter (EKF)**.

It simulates real battery behavior under dynamic load conditions and applies estimation techniques used in battery management systems.

---

## ⚙️ Features

* Lithium-ion battery modeling
* Nonlinear voltage behavior
* EKF-based state estimation
* Noise handling and error analysis

---

## ▶️ How to Run

```bash
python main.py
```

---

## 📊 Example Output

* Estimated SoC closely follows true SoC
* Low estimation error (RMSE calculated)
